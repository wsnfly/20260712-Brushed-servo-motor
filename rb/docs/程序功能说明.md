# 有刷直流伺服电机控制器 — 程序功能说明

## 一、概述

本项目基于 **STM32F103RBT6** 微控制器，实现了一款功能完整的有刷直流伺服电机控制器。支持位置模式、速度模式、位置速度模式、开环模式和外部目标位置模式共 5 种运行模式，配备 MODBUS RTU 工业总线通信接口，可通过 RS485 远程配置和控制。

## 二、硬件架构

| 外设 | 功能 | 引脚 |
|------|------|------|
| TIM1 | 100µs 定时中断（MODBUS超时检测 + PID定时器） | — |
| TIM2 | 10µs 定时中断（脉冲输入高速边沿检测） | — |
| TIM3 | 编码器接口（4倍频模式，64位扩展计数） | PA6=CH1(A相), PA7=CH2(B相) |
| TIM4_CH1 | H桥正向PWM | PB6 |
| TIM4_CH2 | H桥使能1（常开100%） | PB7 |
| TIM4_CH3 | H桥反向PWM | PB8 |
| TIM4_CH4 | H桥使能2（常开100%） | PB9 |
| USART1 | MODBUS RTU 通信 | PA9=TX, PA10=RX |
| PA11 | RS485 方向控制 (DE/RE) | PA11 |
| PB4/PB5 | 可配置多功能输入引脚 | PB4, PB5 |
| PC0 | 保留/备用 | PC0 |

## 三、软件架构

### 3.1 模块划分

```
main.c                  — 主程序入口、外设初始化、主循环
├── motor_control.c/h   — 伺服电机控制核心库
├── modbus.c/h          — MODBUS RTU 从机协议栈
├── afangkaobei.c/h     — 防拷贝保护
└── stm32f1xx_it.c      — 中断服务函数
```

### 3.2 工作模式 (5种)

| 模式 | 值 | 描述 |
|------|-----|------|
| **位置模式** | 0 | PID位置闭环，控制电机精确到达目标位置 |
| **速度模式** | 1 | PID速度闭环，控制电机以目标速度持续运转 |
| **位置速度模式** | 2 | 目标位置按速度累加，走位置PID，实现平滑速度控制 |
| **开环模式** | 3 | 速度值直接映射到PWM输出，无反馈 |
| **外部目标模式** | 4 | PB4/PB5电平选择预设目标位置 |

### 3.3 控制流程

```
TIM1 100µs 中断
  ├── sys_time_100us++          → 系统时间
  ├── MODBUS_OnTimeoutTick()    → MODBUS帧间超时检测
  └── MotorControl_TimerTick100us()
        └── (每50次 ≈ 5ms) MotorControl_Update()
              ├── 编码器速度计算 + 移动平均滤波
              ├── 堵转复位状态机
              ├── 脉冲累加（PB4/PB5输入）
              ├── 位置/速度 PID 运算
              ├── 抗积分饱和
              ├── 限位开关钳制
              └── Motor_SetPWM() → TIM4 占空比

TIM2 10µs 中断
  └── MotorControl_PulseTick10us()
        └── PB4/PB5边沿检测 + 脉冲累加

TIM3 中断
  └── Encoder_OverflowHandler()
        └── 64位编码器计数扩展

主循环
  ├── MotorControl_ProcessInputs()  → 引脚输入处理
  └── MODBUS_Process()             → MODBUS帧解析与响应
```

### 3.4 PID控制算法

- **位置环 PID**：比例 P + 积分 I（过零清零）+ 微分 D（速度负反馈阻尼）
- **速度环 PID**：比例 P + 积分 I（小误差死区）+ 微分 D（速度变化率）
- **抗积分饱和**：输出饱和时回退本次积分累积，防止积分深度饱和
- **速度滤波**：16级移动平均滤波，平滑速度采样值

### 3.5 MODBUS 协议

| 项目 | 说明 |
|------|------|
| 物理层 | RS485 (TTL UART + DE方向控制) |
| 功能码 | 0x03(读), 0x06(写单寄存器), 0x10(写多寄存器) |
| 帧检测 | 3.5字符时间空闲中断检测 |
| CRC16 | 多项式 0xA001, 查表法, 小端序 |
| 控制寄存器 | 0x0000~0x0031 (50个, 可读写) |
| 状态寄存器 | 0x0100~0x0108 (9个, 只读) |
| 波特率 | 在线可切换，支持 9600~115200 |
| 从机地址 | 在线可修改，保存到Flash |

### 3.6 多功能输入引脚 (PB4/PB5)

每个引脚可独立配置为 5 种功能之一：

| 功能 | 值 | 说明 |
|------|-----|------|
| 脉冲输入 | 0 | 每个上升沿改变电机目标位置 |
| 方向信号 | 1 | 脉冲模式下的方向控制 |
| 复位开关 | 2 | 堵转复位时作为原点触发信号 |
| 限位开关 | 3 | 阻止指定方向的运动 |
| 目标电平 | 4 | 外部目标模式下选择预设位置 |

支持极性选择（高电平/上升沿有效 或 低电平/下降沿有效）。

### 3.7 堵转复位功能

- 支持碰撞堵转检测复位
- 可配置复位方向、速度、电流限制、最大搜索距离
- 碰撞后自动后退指定距离
- 支持开机自动复位
- 支持通过MODBUS手动触发复位

### 3.8 参数持久化

所有配置参数（PID、方向、地址、波特率、引脚功能等）自动保存到STM32 Flash最后一页（1KB），上电自动恢复。使用魔数校验确保数据有效性。

## 四、通信接口

### MODBUS 寄存器总览

| 区间 | 数量 | 属性 | 用途 |
|------|------|------|------|
| 0x0000~0x0031 | 50 | 读写 | 控制参数、目标值、配置 |
| 0x0100~0x0108 | 9 | 只读 | 当前状态、位置、速度 |

## 五、保护机制

1. **限位开关保护**：配置为限位功能的引脚激活时，阻止该方向运动
2. **最大运行速度限制**：所有模式下超速自动钳制输出为0
3. **最大输出限制**：PWM最大输出可配置（10~1000）
4. **UART错误恢复**：ORE/FE/NE错误自动清除并重启接收
5. **MODBUS帧堆积保护**：上一帧未处理时丢弃新帧
6. **PID抗积分饱和**：输出饱和时停止积分累积
7. **死区保护**：位置模式误差在死区内停止输出，防止抖动

## 六、调试工具

项目提供 Python 调试工具 `Tools/modbus_debugger.py`，支持：
- 串口连接与参数配置
- 实时监控位置/速度/模式
- PID参数在线调整
- 模式切换与目标值设置
- 通信日志记录
