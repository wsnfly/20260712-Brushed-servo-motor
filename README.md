# MODBUS RTU 使用说明

## 1. 通信参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 接口 | RS485/USART1 | PA9=TX, PA10=RX, PA11=DE(方向控制) |
| 波特率 | 115200 | 支持 9600/19200/38400/57600/115200 |
| 数据格式 | 8N1 | 8数据位, 无校验, 1停止位 |
| 从机地址 | 1 | 范围 1~247, 可通过MODBUS在线修改并保存到Flash |

## 2. 快速开始

### 2.1 连接硬件
- 将控制器 RS485 接口连接到 PC 的 USB-RS485 转换器
- 上位机打开串口调试工具 (波特率 115200, 8N1)

### 2.2 读取电机当前位置
发送: `01 03 01 00 00 04` (读取 0x0100~0x0103 四个寄存器)

响应解析: 返回的 8 字节数据 = 64 位有符号整数（大端序），单位：编码器脉冲数。

### 2.3 让电机转到位置 10000
发送: `01 10 00 02 00 04 08 00 00 00 00 00 00 27 10` 及 CRC

位置值 10000 = 0x00002710, 分 4 个寄存器(每个16位)发送。

### 2.4 让电机以速度 800 脉冲/秒运转
```
01 06 00 00 00 01           → 切换为速度模式
01 10 00 06 00 02 04 00 00 03 20  → 目标速度 = 800
```

## 3. 支持的功能码

| 功能码 | 名称 | 用途 |
|--------|------|------|
| 0x03 | 读保持寄存器 | 读取当前状态、位置、速度等 |
| 0x06 | 写单个寄存器 | 模式切换、单值设置 |
| 0x10 | 写多个寄存器 | 设置64位目标位置、PID参数批量写入 |

## 4. 常用寄存器速查

### 控制寄存器 (可读写 0x0000~0x0031)

| 地址 | 名称 | 说明 |
|------|------|------|
| 0x0000 | 模式设置 | 0=位置, 1=速度, 2=位置速度, 3=开环, 4=外部目标 |
| 0x0001 | 启动方式 | 0=直接启动, 1=标志位启动 |
| 0x0002~0x0005 | 目标位置 | 64位有符号, 4个寄存器拼装(高→低) |
| 0x0006~0x0007 | 目标速度 | 32位有符号, 2个寄存器拼装 |
| 0x0008~0x000A | 位置环PID | Kp/Ki/Kd, ×100存储 |
| 0x000B~0x000D | 速度环PID | Kp/Ki/Kd, ×100存储 |
| 0x000E | 死区 | 脉冲数 |
| 0x000F | 最大输出 | 1~1000 (对应 0.1%~100%) |
| 0x0010 | 启动触发 | 写1启动(标志位模式下), 写0清除启动 |
| 0x0011 | 电机方向 | 0=正转, 1=反转 |
| 0x0012 | 编码器方向 | 0=正常, 1=反转 |
| 0x0014 | 从机地址 | 1~247, 修改后保存到Flash |
| 0x0015 | 波特率 | 实际值/100, 如115200=1152 |
| 0x0016 | 设为零点 | 写1触发, 编码器清零 |
| 0x0017 | 复位模式 | 0=关闭, 1=堵转复位 |
| 0x0020 | 触发复位 | 写1执行一次堵转复位 |
| 0x0021 | 开机自动复位 | 0=关闭, 1=开启 |
| 0x0022~0x0027 | PB4/PB5引脚配置 | 功能/极性/限位方向 |
| 0x0028~0x0029 | 最大运行速度 | 0=无限制 |
| 0x002A~0x0031 | PB4/PB5预设位置 | 64位目标位置 |

### 状态寄存器 (只读 0x0100~0x0108)

| 地址 | 名称 | 说明 |
|------|------|------|
| 0x0100~0x0103 | 当前位置 | 64位有符号, 编码器脉冲数 |
| 0x0104~0x0105 | 当前速度 | 32位有符号, 脉冲/秒 |
| 0x0106 | 当前模式 | 见模式定义 |
| 0x0107 | 状态字 | bit15=启动, bit14=复位中, bit13=复位失败, bit0-3=模式 |
| 0x0108 | 当前PWM | -1000~+1000 |

## 5. 典型应用场景

### 场景一：位置模式点对点定位
```
① 写 0x0001=0          → 直接启动
② 写 0x0000=0          → 位置模式
③ 写 0x0002~0x0005     → 设置目标位置
④ 电机自动运行到目标位置
```

### 场景二：速度模式连续运转
```
① 写 0x0000=1          → 速度模式
② 写 0x0006~0x0007     → 设置目标速度
③ 电机持续运转
④ 写目标速度=0         → 停止
```

### 场景三：位置速度模式(梯形加减速效果)
```
① 写 0x0001=1          → 标志位启动
② 写 0x0000=2          → 位置速度模式
③ 写 0x0006~0x0007     → 设置目标速度
④ 写 0x0010=1          → 启动
⑤ 写目标速度=0         → 停止
```

### 场景四：外部IO控制目标位置
```
① 写 0x0000=4          → 外部目标位置模式
② 配置PB4/PB5功能为4  → 目标位置电平
③ 写 PB4/PB5 预设位置
④ PB4 输入有效 → 到达PB4预设位置
⑤ PB5 输入有效 → 到达PB5预设位置
⑥ 都无效 → 回到原点位置(0)
```

## 6. 注意事项

1. **多字节写入请使用 0x10 功能码**：64位目标位置分4次写入时，固件在全部写完后才生效，避免中途应用半更新数据。

2. **PID参数范围**：写入值 = 实际值 × 100，如 Kp=7.0 则写入 700。负值用16位补码表示。

3. **参数持久化**：PID参数、地址、波特率等修改后自动保存到Flash（每帧最多擦写一次）。

4. **修改从机地址**：写 0x0014 后，响应帧仍用旧地址返回，之后设备立即切换到新地址。

5. **错误恢复**：通信异常时固件会自动清除错误标志并重启接收，无需手动干预。

6. **最大输出保护**：写入范围自动钳位在 10~1000，防止误设为0导致电机不动。

7. **帧间隔**：建议主机发送帧之间保留至少 5ms 间隔，确保固件有足够时间处理。
